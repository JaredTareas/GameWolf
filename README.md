## Descripción

GameWolf es un sistema web para la gestión y venta de videojuegos. 
El sistema permitirá administrar videojuegos, usuarios y pedidos, además de ofrecer a los clientes la posibilidad de consultar el catálogo, realizar compras y dar seguimiento a sus pedidos.
